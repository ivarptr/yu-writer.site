RELEASE NOTES

Alpha 0.3.0

2017-06-15
  • 增加了数学公式支持。
  • 插入链接窗口增加了罗列文档和章节功能。
  • 插入代码窗口增加了格式化（美化）代码功能。
  • 大纲面板增加了章节左缩进和右缩进功能。
  • 插入链接、代码、数学公式等窗口均增加了更新功能，比如可更新文档中已存在的链接。

Alpha 0.2.1

2017-05-30
  • 修复了上一个版本引入的一些 BUG。
  • 增加了手动检测新版本的功能。
  • 改进了隐藏工具栏时的UI。

Alpha 0.2.0

2017-05-19
  • 增加了打开、创建库外文档的功能，现在可以把 Yu Writer 作为系统的 Markdown 和纯文本编辑器使用了。
  • 增加 Tag 标签管理和搜索功能。
  • 文档的编码、换行符自动识别。
  • 增加 YAML 格式的文档 Front Matter 支持。
  • 解决了中文输入法光标位置乱跳的问题。

Alpha 0.1.2

2017-03-28
完善了搜索相关的功能，改进一些界面上的外观。
  • 增加了全文搜索功能。
  • 增加全库范围搜索功能。
  • 增加大纲搜索功能。
  • 改进搜索栏的外观、默认隐藏标签栏。

Alpha 0.1.1

2017-03-18
添加了自动检测新版本的功能，修正一些比较明显的 bug。
  • 修复了导出到 PDF 文件巨大的 BUG。 感谢 @ZN5757 反馈。
  • 改进新建文档的提示。 上一个版本因为新安装应用时默认没有选中任何一个库， 所以一开始创建不了文档，提示也不够清晰，给人的体验不好。 新建文档时假如文档库的位置不正确会有明确的提示。 感谢 @linglingfa, @bluebird0605 反馈。
  • 工具栏上的 “Dashboard” 按钮名称已改为更准确的 “File”。 在 “File” 菜单下添加了常用的 “New Library”、“New Folder” 和 “Search” 菜单项。
  • 修复了关闭应用窗口后，点击 Dock 上的应用图标不会新建窗口的 BUG。 感谢 @ZN5757， @bitweng 反馈。
  • “Help” 菜单的 “Manual” 链接到 在线的说明文档。
  • 修复了切换多个 Yu Writer 窗口时菜单的状态没有正确更新的 BUG。
  • 新增了 Card 模板和 Candy 样式集。for Candy Hu

Alpha 0.1.0

2017-03-15
Yu Writer 开发了近一年多时间，因为我不是专业 程序员只能在平时空闲时填坑，加上今年比较忙， 要把预期目标所有功能完成可能还得需一年时间，所以先释放一个 Alpha 测试版出来，供朋友们测试、体验、提意见，顺便看看反响如何，我填坑时也不至于太闷 😄。 所以，感谢你来到这个页面！感谢你的参与和支持！
